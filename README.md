# Daily Activity Flutter App

A simple Flutter starter app that shows a user's daily phone activity in a clean mobile dashboard.

## Features

- Daily overview card with top activity and phone pickups
- Quick day selector for recent activity snapshots
- Metrics for screen time, steps, focus time, and pickups
- Activity category breakdown with progress bars
- Simple widget test for the main dashboard

## Run locally

If you want Flutter to generate the missing platform folders (`android/`, `ios/`, etc.) inside this minimal starter, run:

```bash
flutter create .
flutter pub get
flutter run
```

## Test

```bash
flutter test
```
